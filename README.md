# Convert-ClubspotScoringPanel
A simple parser to convert Clubspot scoring panel data into CSV.

.\Convert-ClubspotScoringPanel.ps1 -importfile input.txt -columns 10 >output.csv

    -importfile mandatory, the input file captured by cutting-and-pasting the scoring panel into a text file
    -columns mandatory, integer number of columns displayed in the scoring panel captured
